gyroid
======

**gyroid** is a python package that generates *symmetry adapted basis functions* based on the space group of a unit cell. 

Quickstart
----------

1. Install
^^^^^^^^^^

::

    $ easy_install gyroid

or

::

    $ tar -xvf gyroid-xxx.tar.gz
    $ cd gyroid-xxx
    $ python setup.py install

2. Usage
^^^^^^^^

::

    >>>import gyroid as gy
    >>>gy.make_basis("p6mm")
